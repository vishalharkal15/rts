## One-Click Deploy

1. Click the "Deploy on Railway" button above
2. Railway will automatically:
   - Build the Docker container
   - Deploy your application
   - Provide a public URL

## Manual Deploy

```bash
# Install Railway CLI
npm i -g @railway/cli

# Login
railway login

# Deploy
railway init
railway up
```
